### 3. Backend Docker Service Enhancements

**30+ New Methods:**

**Container Management:**
- `getContainerStats()` - Real-time resource monitoring
- `getContainerResourceLimits()` - View limits
- `setContainerResourceLimits()` - Set memory/CPU limits

**Volume Operations:**
- `listVolumes()` - List all volumes
- `createVolume()` - Create with custom drivers
- `deleteVolume()` - Remove volumes
- `getVolumeInfo()` - Inspect details

**Network Operations:**
- `listNetworks()` - List all networks
- `createNetwork()` - Create custom networks
- `deleteNetwork()` - Remove networks
- `connectContainerToNetwork()` - Connect containers
- `disconnectContainerFromNetwork()` - Disconnect

**Image Operations:**
- `listImages()` - List with sizes
- `deleteImage()` - Remove specific image
- `pruneImages()` - Clean unused images
- `pullImage()` - Pull from registries

**Docker Compose:**
- `deployWithCompose()` - Multi-container deployment
- `stopCompose()` - Stop all services
- `getComposeStatus()` - Service status

**Execution:**
- `execInContainer()` - Run commands
- `getContainerProcesses()` - View processes

**Backup & Restore:**
- `exportContainer()` - Create snapshots
- `saveImageToFile()` - Export to tar
- `loadImageFromFile()` - Import from tar

**Registry:**
- `registryLogin()` - Authenticate
- `pushImage()` - Push to registry
- `tagImage()` - Tag for registry

**System:**
- `getSystemInfo()` - Docker info
- `systemPrune()` - System cleanup
- `getDiskUsage()` - Disk analysis
